# 逻辑连接符 `if` 

```follow
term prop if(prop p0, prop p1, prop p2) { if ( p0, p1, p2 ) }
axiom if.def.1(prop p0, prop p1, prop p2) {
  |- iff(if(p0, p1, p2), or(and(p0, p1), and(not(p0), p2)))
}
```
