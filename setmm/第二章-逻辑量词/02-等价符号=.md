
# 等价符号 `=` 

```follow
term prop eq(class c0, class c1) { ( c0 = c1 )}
```

```follow
axiom a6(set s0, set s1) {
  |- not(forall(s0, not(eq(stoc(s0), stoc(s1)))))
  |- exist(s0, eq(stoc(s0), stoc(s1)))
  diff (s0, s1)
}
```

## `exist.gen` 

```follow
thm exist.gen(set s0, prop p0) {
  |- exist(s0, p0)
  -| p0
} = {
  mp(exist(s0,p0), exist(s0,eq(stoc(s0),stoc(hs0))))
  a6(s0, hs0)
  a4.existi.gen(s0, eq(stoc(s0),stoc(hs0)), p0)
  a1i(eq(stoc(s0),stoc(hs0)), p0)
}
```

```follow
thm forall.2exist(set s0, prop p0) {
```


## Specialization

```follow
thm forall.special.1(set s0, set s1, prop p0, prop p1) {
  |- imp(forall(s0, p0), p1)
  -| imp(not(p1), forall(s0, not(p1)))
  -| imp(eq(stoc(s0), stoc(s1)), imp(p0, p1))
  diff (s0, s1)
} = {
  mp(imp(forall(s0,p0),p1), exist(s0,eq(stoc(s0),stoc(s1))))
  a6(s0, s1)
  rw3(exist(s0,eq(stoc(s0),stoc(s1))), forall(s0,p0), p1, not(forall(s0,not(p1))))
  con2i(forall(s0,not(p1)), p1)
  rw3(exist(s0,eq(stoc(s0),stoc(s1))), forall(s0,p0), not(forall(s0,not(p1))), exist(s0,p1))
  exist.def(s0, p1)
  syl(exist(s0,eq(stoc(s0),stoc(s1))), imp(forall(s0,p0),exist(s0,p1)), exist(s0,imp(p0,p1)))

}
```
