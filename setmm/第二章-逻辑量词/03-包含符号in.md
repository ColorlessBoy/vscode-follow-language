
# 包含符号 `in` 


```follow
term prop in(class c0, class c1) { (c0 ∈ c1) }
```

```follow
axiom a8.1(set s0, set s1, set s2) {
  |- imp(eq(s2c(s0), s2c(s1)), imp(in(s2c(s0), s2c(s2)), in(s2c(s1), s2c(s2))))
  diff (s0, s1) (s0, s2)
}
```

```follow
axiom a8.2(set s0, set s1, set s2) {
  |- imp(eq(s2c(s0), s2c(s1)), imp(in(s2c(s0), s2c(s2)), in(s2c(s1), s2c(s2))))
  diff (s1, s0) (s1, s2)
}
```
